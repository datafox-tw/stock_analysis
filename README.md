# 📈 Stock Analysis (Modernized Version: 2026-03-ver)

> [!IMPORTANT]
> **保留初衷，進化形式**。
> 這個版本並非要否定三年前初學 Python 時的自己，而是為了將當時最珍貴的資產——核心邏輯與親手構建的程式架構——用更現代、更穩健的方式重新包裝。
>
> 當時雖然「代碼寫得很爛」，但那份「從零開始解決問題、實現資料爬取與策略計算」的心路歷程是無可取代的。這個版本旨在保留那份精髓，同時提升其作為一個「真正專案」的專業度。

---

## 📅 專案演進史

- **2021年12月**：第一次接觸 Python。建立核心策略邏輯。
- **2023年4月**：建立 Flask 初版。
- **2026年3月**：現代化升級。
  - **核心保留**：維持原有的資料爬取規則、指標計算公式與回測演算法。
  - **架構重構**：優化資料傳遞機制（取代脆弱的隱藏欄位字串傳遞）、模組化核心邏輯。
  - **介面升級**：採用現代化的前端框架與設計語彙，提升使用者體驗。
---

## 🛠️ 技術棧與特色

- **核心語言**：Python 3
- **前端框架**：Flask
- **數據處理**：Pandas, CSV
- **爬蟲工具**：Requests, urllib3
- **策略回測**：
  - **KD 指標**：可自定義買入/賣出門檻。
  - **威廉指標 (William %R)**：支援 14 日與 28 日參數。
  - **MA 均線策略**：收錄了 4 種經典的均線交叉與排列策略。

---

## 🚀 如何運行

1. **安裝依賴**：
   ```bash
   pip install -r requirements.txt
   ```

2. **啟動 Flask 應用**：
   ```bash
   python app.py
   ```

3. **訪問網站**：
   在瀏覽器中開啟 `http://127.0.0.1:5000`。

## 📁 檔案說明

- `app.py`: 2023 年開發的 Flask 網頁版主程式。
- `original(no_frontend).py`: 2021 年最初開發的腳本版（純命令行介面）。
- `stocklist.csv`: 台灣上市櫃股票清單數據（用於名稱查詢）。
- `templates/`: 存放 HTML 模板。
- `static/`: 存放 CSS 樣式表。

##  各式各樣實務限制
1. 每次股票輸入區間範圍之後只能夠選擇一次策略按鈕 之後要再重新輸入一次區間範圍和股票
2. 下載檔案時前端不會顯示正在下載而是啥都不動 
3. 完全沒有error handling機制
